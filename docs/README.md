# 구현할 기능 목록

### 1. 로또 구입 및 번호 발행
- [ ] 로또 구입 금액 입력
    - 예외 에러 처리
- [ ] 금액에 따라 로또 발행
    - 1~45 사이의 서로 다른 6개의 숫자 랜덤 생성
    - 각 로또 내 숫자 오름차순으로 저장

### 2. 당첨 번호 및 보너스 번호 확정(입력)
- [ ] 당첨 번호 입력 및 저장
    - 1~45 사이의 서로 다른 6개의 숫자
    - 번호 쉼표(`,`)로 구분 / 띄어쓰기 X
    - 잘못된 입력에 대한 예외처리
- [ ] 보너스 번호 입력 및 저장
    - 당첨 번호와 중복되지 않는 번호 입력
    - 잘못된 입력에 대한 예외처리

### 3. 당첨 결과 계산
- [ ] 당첨 통계 계산 및 출력
- [ ] 당첨금액 및 구입 금액으로 수익률 계산
